# Peek at a slide

Displays a thumbnail of the current slide

## Usage

``` r
peek(x, ...)

## S7 method for class <r2slides::slide>
peek(x, ...)
```

## Arguments

- x:

  A [slide](https://adam-hubbs.github.io/r2slides/reference/slide.md)
  object.

- ...:

  Ignored; present for S7 generic compatibility.

- slide:

  The slide object you want to get an image of
