# Chart ID class

A typed reference to a specific chart within a Google Sheets sheet.
Inherits from
[sht_id](https://adam-hubbs.github.io/r2slides/reference/sht_id.md).

## Usage

``` r
chart_id(
  spreadsheet_id = character(0),
  sheet_id = character(0),
  chart_id = character(0)
)
```

## Arguments

- spreadsheet_id:

  A single string. The Google Sheets spreadsheet ID.

- sheet_id:

  A single string. The sheet tab ID.

- chart_id:

  A single string. The chart ID.
