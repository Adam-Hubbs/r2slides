# Create a new Google Sheets spreadsheet

Create a new Google Sheets spreadsheet

## Usage

``` r
new_spreadsheet(title = "Untitled Spreadsheet", set_active = TRUE)
```

## Arguments

- title:

  Optional. A single string. The title of the spreadsheet.

- set_active:

  Optional. A logical value indicating whether to set the spreadsheet as
  the active spreadsheet. Default: `TRUE`.

## Value

A `spreadsheet` object
