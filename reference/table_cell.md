# A single cell in an r2slides_table

A single cell in an r2slides_table

## Usage

``` r
table_cell(
  row_index = integer(0),
  col_index = integer(0),
  text = NULL,
  style = NULL
)
```

## Arguments

- row_index:

  0-based integer row index.

- col_index:

  0-based integer column index.

- text:

  Character string with the cell's text content, or \`NULL\`.

- style:

  A \[cell_style\] object, or \`NULL\`.
