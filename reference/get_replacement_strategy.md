# Get the current replacement strategy

Returns the value of the `r2slides_replacement_strategy` environment
variable, defaulting to `"add"` if unset.

## Usage

``` r
get_replacement_strategy()
```

## Value

A string: `"add"`, `"replace"`, or `"skip"`.
