# Visualize a color

Renders a color swatch for
[solid_color](https://adam-hubbs.github.io/r2slides/reference/r2s_color.md)
objects, or prints an informative message for
[theme_color](https://adam-hubbs.github.io/r2slides/reference/r2s_color.md)
objects whose actual value depends on the presentation theme.

## Usage

``` r
visualize_color(color, ...)
```

## Arguments

- color:

  An
  [r2s_color](https://adam-hubbs.github.io/r2slides/reference/r2s_color.md)
  object.

- ...:

  Ignored; present for S7 generic compatibility.
