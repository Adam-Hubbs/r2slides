# Creates a slide_position object with defaults in the top left of the slide

Creates a slide_position object with defaults in the top left of the
slide

## Usage

``` r
in_top_left(
  convert_slide_size = FALSE,
  slide_size_old = NULL,
  slide_size = c(5.625, 10)
)

in_top_middle(
  convert_slide_size = FALSE,
  slide_size_old = NULL,
  slide_size = c(5.625, 10)
)

in_top_right(
  convert_slide_size = FALSE,
  slide_size_old = NULL,
  slide_size = c(5.625, 10)
)

in_bottom_left(
  convert_slide_size = FALSE,
  slide_size_old = NULL,
  slide_size = c(5.625, 10)
)

in_bottom_middle(
  convert_slide_size = FALSE,
  slide_size_old = NULL,
  slide_size = c(5.625, 10)
)

in_bottom_right(
  convert_slide_size = FALSE,
  slide_size_old = NULL,
  slide_size = c(5.625, 10)
)
```

## Arguments

- convert_slide_size:

  A logical. Optional.

- slide_size_old:

  Old Slide size specifications. Optional. Used for conversion between
  powerpoint and google slides sizes

- slide_size:

  New Slide size specifications. Optional.
