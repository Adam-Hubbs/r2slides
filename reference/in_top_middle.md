# Creates a slide_position object with defaults in the top middle of the slide

Creates a slide_position object with defaults in the top middle of the
slide

## Usage

``` r
in_top_middle(
  convert_slide_size = FALSE,
  slide_size_old = NULL,
  slide_size = c(5.625, 10)
)
```

## Arguments

- convert_slide_size:

  A logical. Optional.

- slide_size_old:

  Old Slide size specifications. Optional. Used for conversion between
  powerpoint and google slides sizes

- slide_size:

  New Slide size specifications. Optional.
