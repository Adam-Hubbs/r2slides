# Sheet ID class

A typed reference to a specific sheet (tab) within a Google Sheets
spreadsheet.

## Usage

``` r
sht_id(spreadsheet_id = character(0), sheet_id = character(0))

is_sht_id(x)
```

## Arguments

- spreadsheet_id:

  A single string. The Google Sheets spreadsheet ID.

- sheet_id:

  A single string. The sheet tab ID.

- x:

  An object to test
